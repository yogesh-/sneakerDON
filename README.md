# SNEAKERS

SNEAKERS is an ecommerce shop built for the  sneakers universe.

The Ecommmerce web App is divided among the following pages

- Homepage
- Product Lists pages
- Wishlist
- Cart
- Login
- Logout

URL - sneaker-don.netlify.app
